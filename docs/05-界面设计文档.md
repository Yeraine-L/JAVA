# 校园流浪猫狗投喂与领养管理系统 - 界面设计文档

## 1. 界面设计目标

本项目界面采用 Java Swing 实现，整体遵循简洁、统一、易演示的原则，主界面采用左侧导航栏 + 右侧内容区布局，所有新增、编辑、申请、审核类操作统一使用弹窗完成。

## 2. 全局界面规范

- 主窗口支持最大化、最小化
- 左侧导航固定，右侧内容可滚动
- 所有表格统一使用 JTable
- 所有提示统一使用消息弹窗
- 所有确认类操作统一使用确认对话框
- 图片加载失败显示默认占位图
- 表单标题、按钮样式、字体大小尽量统一

## 4. 详细页面设计

### 4.1 UI01 登录窗口

组件：

- 用户名输入框
- 密码输入框
- 角色提示文本
- 登录按钮
- 注册按钮
- 退出按钮

交互：

- 用户名或密码为空时提示
- 登录成功后按角色跳转主界面
- 注册按钮打开注册弹窗

### 4.2 UI02 注册弹窗

组件：

- 用户名输入框
- 密码输入框
- 手机号输入框
- 身份类型下拉框
- 确认按钮
- 取消按钮

校验：

- 用户名唯一
- 密码不少于 6 位
- 手机号必须 11 位数字

### 4.3 UI03 主窗口

布局：

- 顶部：系统标题、当前登录用户、角色、退出登录按钮
- 左侧：导航菜单
- 右侧：内容切换区域

普通用户导航建议：

- 动物信息
- 校园地图
- 我的申请
- 消息通知
- 个人中心
- 公告查看

志愿者导航建议：

- 动物信息
- 校园地图
- 投喂点申请
- 投喂记录
- 异常上报

- 消息通知
- 个人中心
- 公告查看

管理员导航建议：

- 动物管理

## 5. 弹窗统一规范

- 标题显示明确业务名称
- 大小固定，避免拉伸变形
- 底部统一为确认和取消按钮
- 成功、失败、警告提示统一样式
