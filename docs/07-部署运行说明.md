# 校园流浪猫狗投喂与领养管理系统 - 部署运行说明

## 1. 项目运行环境

- 操作系统：Windows
- JDK：1.8 及以上
- 数据库：MySQL 8.0
- IDE：IntelliJ IDEA

## 2. 运行前准备

### 2.1 安装 JDK

- 安装 JDK 8 或更高版本
- 配置 JAVA_HOME
- 确保 java -version 可正常执行

### 2.2 安装 MySQL

- 安装 MySQL 8.0
- 创建数据库 campus_pet_system
- 导入项目提供的 SQL 脚本

### 2.3 导入依赖

需要准备以下 jar 包：

- MySQL JDBC 驱动
- JFreeChart
- JCommon

如果项目使用 IDEA 管理 jar 包，可统一放在 lib 目录后添加到项目依赖。

## 3. 数据库初始化步骤

1. 打开 MySQL 客户端
2. 创建数据库 campus_pet_system
3. 执行项目中的初始化脚本
4. 检查核心表是否创建成功
5. 检查管理员与测试数据是否已导入

## 4. 配置说明

项目中需要维护数据库配置：

- 数据库地址
- 数据库端口
- 数据库名
- 用户名
- 密码

建议放在 config.properties 中。

## 5. 启动方式

### 方式一：通过 IDEA 启动

1. 用 IDEA 打开项目
2. 配置 JDK
3. 配置依赖 jar
4. 找到 Main.java
5. 直接运行主类

### 方式二：打包后运行

1. 将项目打成 jar 包
2. 确保依赖 jar 可用
3. 双击运行或通过命令行执行 java -jar

## 6. 默认账号

- 管理员：admin / admin123
- 普通用户：user01 / 123456
- 志愿者：vol01 / 123456

## 7. 首次演示建议流程

1. 先使用管理员登录
2. 演示动物管理与公告管理
3. 退出后使用普通用户登录
4. 演示查看动物与提交领养申请
5. 切回管理员审核申请
6. 再切回普通用户查看审核结果
7. 最后使用志愿者演示投喂管理和异常上报

## 8. 常见问题处理

### 8.1 数据库连接失败

- 检查 MySQL 是否启动
- 检查用户名密码是否正确
- 检查数据库名是否正确
- 检查 JDBC 驱动是否已导入

### 8.2 图片无法显示

- 检查图片路径是否存在
- 检查默认占位图是否存在
- 检查资源目录路径是否正确

### 8.3 图表无法显示

- 检查 JFreeChart 依赖是否导入
- 检查统计数据是否为空

### 8.4 项目无法启动

- 检查 JDK 配置
- 检查主类是否正确
- 检查项目依赖是否缺失
