# 用户中心模块开发文档

## 1. 页面与弹窗清单

### 用户中心主界面
- **文件**：`src/main/java/com/campuspet/view/user/UserCenterPanel.java`
- **功能**：展示用户个人信息、领养申请列表、消息通知入口
- **布局**：顶部个人信息区域 + 底部功能卡片区域

### 修改密码弹窗
- **文件**：`src/main/java/com/campuspet/view/user/ChangePasswordDialog.java`
- **功能**：用户修改密码
- **布局**：旧密码输入框 + 新密码输入框 + 确认密码输入框 + 确认/取消按钮

### 我的领养申请列表
- **文件**：`src/main/java/com/campuspet/view/user/MyApplyListPanel.java`
- **功能**：展示当前用户的所有领养申请记录
- **布局**：表格展示 + 状态标签 + 刷新按钮

### 领养回访查看
- **文件**：`src/main/java/com/campuspet/view/user/AdoptFeedbackPanel.java`
- **功能**：查看已领养宠物的回访记录
- **布局**：表格展示 + 回访详情查看

## 6. 界面层开发

### UserCenterPanel.java

**功能**：
- 显示用户头像/名称
- 显示用户角色、注册时间
- 提供修改密码、查看申请、查看回访等功能入口

**布局**：
```

├─────────────────────────────────────────┤

│  │修改密码 │ │我的申请 │ │消息中心 │  │
│  │         │ │         │ │         │  │
│  └─────────┘ └─────────┘ └─────────┘  │
│  ┌─────────┐                            │
│  │领养回访 │                            │
│  └─────────┘                            │
└─────────────────────────────────────────┘
```

### ChangePasswordDialog.java

**功能**：
- 输入旧密码
- 输入新密码
- 确认新密码
- 提交修改

**校验**：
- 旧密码正确性
- 新密码与确认密码一致性
- 密码强度校验

### MyApplyListPanel.java

**功能**：
- 显示用户的所有领养申请
- 表格展示：宠物名称、申请时间、状态、审核意见
- 刷新按钮

### AdoptFeedbackPanel.java

**功能**：
- 显示已领养宠物的回访记录
- 表格展示：回访时间、回访内容、回访结果

## 7. 绑定事件逻辑

### UserCenterPanel.java
- 修改密码按钮 → 打开 ChangePasswordDialog
- 我的申请按钮 → 打开 MyApplyListPanel
- 消息中心按钮 → 打开消息中心（待开发）
- 领养回访按钮 → 打开 AdoptFeedbackPanel

### ChangePasswordDialog.java
- 确认按钮 → 调用 UserService.updatePassword()
- 取消按钮 → 关闭弹窗

## 8. 表单校验

### ChangePasswordDialog.java
- 旧密码不能为空
- 新密码不能为空且长度 ≥ 6 位
- 确认密码必须与新密码一致
- 旧密码必须正确（查询数据库验证）

## 9. 正常流程测试

1. 用户登录后进入用户中心
2. 点击"修改密码"按钮
3. 输入旧密码、新密码、确认密码
4. 点击确认，密码修改成功
5. 点击"我的申请"按钮
6. 查看领养申请列表
7. 点击"领养回访"按钮
8. 查看回访记录

## 10. 异常流程测试

1. 修改密码时旧密码错误 → 提示"旧密码错误"
2. 新密码与确认密码不一致 → 提示"两次密码不一致"
3. 新密码长度不足 → 提示"密码长度至少6位"
4. 数据库操作失败 → 提示"操作失败，请重试"
