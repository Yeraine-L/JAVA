# 公告管理模块开发文档

## 1. 页面与弹窗清单

### 公告列表面板（管理员）
- **文件**：`src/main/java/com/campuspet/view/admin/AnnouncementListPanel.java`
- **功能**：展示所有公告，支持编辑、删除
- **布局**：表格展示 + 新增/编辑/删除按钮

### 公告发布弹窗（管理员）
- **文件**：`src/main/java/com/campuspet/view/admin/PublishAnnouncementDialog.java`
- **功能**：发布新公告
- **布局**：标题输入框 + 内容文本域 + 发布按钮

### 公告编辑弹窗（管理员）
- **文件**：`src/main/java/com/campuspet/view/admin/EditAnnouncementDialog.java`
- **功能**：编辑现有公告
- **布局**：标题输入框 + 内容文本域 + 保存按钮

### 公告列表面板（用户）
- **文件**：`src/main/java/com/campuspet/view/user/UserAnnouncementPanel.java`
- **功能**：展示所有公告（只读）
- **布局**：列表展示 + 公告详情查看

### 公告详情弹窗（用户）
- **文件**：`src/main/java/com/campuspet/view/user/AnnouncementDetailDialog.java`
- **功能**：查看公告详细内容
- **布局**：标题 + 内容 + 发布时间 + 发布人

## 2. 数据库表与字段

### announcement 表（已有）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| title | VARCHAR(100) | 标题 |
| content | TEXT | 内容 |
| publisher_id | BIGINT | 发布人ID |
| publisher_name | VARCHAR(50) | 发布人姓名 |
| publish_time | DATETIME | 发布时间 |
| update_time | DATETIME | 更新时间 |

## 6. 界面层开发

### AnnouncementListPanel.java（管理员）

**功能**：
- 表格展示所有公告
- 显示：标题、发布时间、发布人
- 新增按钮 → 打开 PublishAnnouncementDialog
- 编辑按钮 → 打开 EditAnnouncementDialog
- 删除按钮 → 删除公告

### PublishAnnouncementDialog.java（管理员）

**功能**：
- 输入公告标题
- 输入公告内容
- 点击发布按钮

**校验**：
- 标题不能为空
- 内容不能为空

### EditAnnouncementDialog.java（管理员）

**功能**：
- 显示当前公告标题和内容
- 修改标题和内容
- 保存修改

### UserAnnouncementPanel.java（用户）

**功能**：
- 列表展示所有公告
- 点击公告查看详情
- 按发布时间倒序排列

### AnnouncementDetailDialog.java（用户）

**功能**：
- 显示公告标题
- 显示公告内容
- 显示发布时间和发布人

## 7. 绑定事件逻辑

### AnnouncementListPanel.java（管理员）
- 新增按钮 → 打开 PublishAnnouncementDialog
- 编辑按钮 → 打开 EditAnnouncementDialog（传入公告ID）
- 删除按钮 → 调用 AnnouncementService.delete()
- 刷新按钮 → 重新加载公告列表

### UserAnnouncementPanel.java（用户）
- 列表项点击 → 打开 AnnouncementDetailDialog（传入公告ID）
- 刷新按钮 → 重新加载公告列表

## 8. 表单校验

### PublishAnnouncementDialog.java
- 标题不能为空
- 内容不能为空

### EditAnnouncementDialog.java
- 标题不能为空
- 内容不能为空

## 9. 正常流程测试

1. 管理员登录后进入公告管理
2. 点击"新增"按钮
3. 输入标题和内容，点击发布
4. 公告发布成功，列表刷新
5. 点击"编辑"按钮
6. 修改标题和内容，点击保存
7. 公告更新成功，列表刷新
8. 点击"删除"按钮
9. 确认删除，公告删除成功
10. 普通用户登录后进入公告查看
11. 点击公告查看详情
12. 公告详情正确显示

## 10. 异常流程测试

1. 发布公告时标题为空 → 提示"标题不能为空"
2. 发布公告时内容为空 → 提示"内容不能为空"
3. 编辑公告时标题为空 → 提示"标题不能为空"
4. 编辑公告时内容为空 → 提示"内容不能为空"
5. 删除不存在的公告 → 提示"公告不存在"
6. 数据库操作失败 → 提示"操作失败，请重试"
