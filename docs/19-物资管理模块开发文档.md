# 物资管理模块开发文档

## 1. 页面与弹窗清单

### 物资列表面板（管理员）
- **文件**：`src/main/java/com/campuspet/view/admin/GoodsListPanel.java`
- **功能**：管理员查看所有物资库存
- **布局**：表格展示 + 入库/出库按钮

### 物资入库面板（管理员）
- **文件**：`src/main/java/com/campuspet/view/admin/GoodsInPanel.java`
- **功能**：管理员进行物资入库操作
- **布局**：选择物资 + 输入数量 + 入库按钮

### 物资出库面板（管理员）
- **文件**：`src/main/java/com/campuspet/view/admin/GoodsOutPanel.java`
- **功能**：管理员进行物资出库操作
- **布局**：选择物资 + 输入数量 + 出库按钮

### 物资记录列表面板（管理员）
- **文件**：`src/main/java/com/campuspet/view/admin/GoodsRecordListPanel.java`
- **功能**：管理员查看物资出入库记录
- **布局**：表格展示 + 刷新按钮

## 2. 数据库表与字段

### goods 表（新建）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| name | VARCHAR(100) | 物资名称 |
| type | VARCHAR(50) | 物资类型（食品/药品/用品） |
| unit | VARCHAR(20) | 单位（袋/盒/瓶/箱） |
| total_quantity | INT | 总数量 |
| min_quantity | INT | 最低库存预警阈值 |
| create_time | DATETIME | 创建时间 |

### goods_record 表（新建）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| goods_id | BIGINT | 物资ID |
| goods_name | VARCHAR(100) | 物资名称 |
| type | VARCHAR(20) | 类型（入库/出库） |
| quantity | INT | 数量 |
| operator_id | BIGINT | 操作人ID |
| operator_name | VARCHAR(50) | 操作人姓名 |
| remark | VARCHAR(255) | 备注 |
| create_time | DATETIME | 创建时间 |

## 6. 界面层开发

### GoodsListPanel.java（管理员）

**功能**：
- 表格展示所有物资
- 显示：名称、类型、单位、总数量、最低库存、状态（正常/预警）
- 入库按钮 → 打开 GoodsInPanel
- 出库按钮 → 打开 GoodsOutPanel
- 刷新按钮

**状态显示**：
- 总数量 ≤ 最低库存 → 红色显示"预警"
- 总数量 > 最低库存 → 绿色显示"正常"

### GoodsInPanel.java（管理员）

**功能**：
- 选择物资（下拉框）
- 输入入库数量
- 输入备注（可选）
- 入库按钮

**校验**：
- 物资不能为空
- 入库数量必须 > 0

### GoodsOutPanel.java（管理员）

**功能**：
- 选择物资（下拉框）
- 输入出库数量
- 输入备注（可选）
- 出库按钮

**校验**：
- 物资不能为空
- 出库数量必须 > 0
- 出库数量 ≤ 当前库存

### GoodsRecordListPanel.java（管理员）

**功能**：
- 表格展示所有出入库记录
- 显示：物资名称、类型、数量、操作人、时间、备注
- 刷新按钮

## 7. 绑定事件逻辑

### GoodsListPanel.java（管理员）
- 入库按钮 → 打开 GoodsInPanel（传入选中物资ID）
- 出库按钮 → 打开 GoodsOutPanel（传入选中物资ID）
- 刷新按钮 → 重新加载物资列表

### GoodsInPanel.java（管理员）
- 入库按钮 → 调用 GoodsService.inStock()

### GoodsOutPanel.java（管理员）
- 出库按钮 → 调用 GoodsService.outStock()

### GoodsRecordListPanel.java（管理员）
- 刷新按钮 → 重新加载出入库记录列表

## 8. 表单校验

### GoodsInPanel.java
- 物资不能为空
- 入库数量必须 > 0

### GoodsOutPanel.java
- 物资不能为空
- 出库数量必须 > 0
- 出库数量 ≤ 当前库存

## 10. 正常流程测试

1. 管理员登录后进入物资管理
2. 查看物资列表
3. 点击"入库"按钮
4. 选择物资，输入数量，点击入库
5. 物资入库成功，库存增加
6. 点击"出库"按钮
7. 选择物资，输入数量，点击出库
8. 物资出库成功，库存减少
9. 查看出入库记录列表
10. 记录正确显示

## 11. 异常流程测试

1. 入库时物资为空 → 提示"请选择物资"
2. 入库时数量 ≤ 0 → 提示"入库数量必须 > 0"
3. 出库时物资为空 → 提示"请选择物资"
4. 出库时数量 ≤ 0 → 提示"出库数量必须 > 0"
5. 出库时数量 > 库存 → 提示"库存不足"
6. 数据库操作失败 → 提示"操作失败，请重试"

## 12. 库存预警

- 当物资总数量 ≤ 最低库存时，表格中显示红色"预警"状态
- 可以单独查看低库存物资列表
