# 数据库导入说明

## 1. 确保 MySQL 已安装并启动

在命令行中运行以下命令检查 MySQL 是否运行：
```bash
mysql -u root -p
```

如果提示连接失败，请先启动 MySQL 服务。

## 2. 导入数据库脚本

### 方式一：命令行导入

```bash
mysql -u root -p < src/main/resources/db/init.sql
```

### 方式二：MySQL 客户端导入

1. 打开 MySQL 客户端（如 MySQL Workbench、Navicat 等）
2. 连接到数据库（用户名：root，密码：root）
3. 执行 `src/main/resources/db/init.sql` 文件

### 方式三：手动执行 SQL

1. 打开 MySQL 客户端
2. 依次执行以下 SQL 语句：

```sql
-- 1. 创建数据库
CREATE DATABASE IF NOT EXISTS campus_pet_system CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci;

-- 2. 使用数据库
USE campus_pet_system;

-- 3. 创建用户表
CREATE TABLE IF NOT EXISTS users (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    username VARCHAR(50) NOT NULL UNIQUE,
    password VARCHAR(100) NOT NULL,
    phone VARCHAR(20) NOT NULL,
    role VARCHAR(20) NOT NULL,
    status VARCHAR(20) NOT NULL DEFAULT '正常',
    create_time DATETIME NOT NULL
);

-- 4. 创建动物表
CREATE TABLE IF NOT EXISTS pets (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    pet_no VARCHAR(50) NOT NULL UNIQUE,
    name VARCHAR(50),
    type VARCHAR(20) NOT NULL,
    gender VARCHAR(10),
    age VARCHAR(20),
    photo VARCHAR(255),
    location VARCHAR(100),
    area VARCHAR(100),
    health VARCHAR(20),
    personality VARCHAR(20),
    rescue_story TEXT,
    status VARCHAR(20) NOT NULL,
    map_x INT,
    map_y INT,
    create_time DATETIME NOT NULL,
    update_time DATETIME NOT NULL
);

-- 5. 创建领养申请表
CREATE TABLE IF NOT EXISTS adopt_apply (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    user_id BIGINT NOT NULL,
    pet_id BIGINT NOT NULL,
    applicant_name VARCHAR(50) NOT NULL,
    phone VARCHAR(20) NOT NULL,
    address VARCHAR(255) NOT NULL,
    experience VARCHAR(255) NOT NULL,
    reason TEXT,
    plan TEXT,
    apply_time DATETIME NOT NULL,
    status VARCHAR(20) NOT NULL,
    review_opinion VARCHAR(255),
    verify_record VARCHAR(255),
    agreement_signed VARCHAR(20) DEFAULT '未签署',
    review_time DATETIME
);

-- 6. 创建公告表
CREATE TABLE IF NOT EXISTS announcement (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    title VARCHAR(100) NOT NULL,
    content TEXT NOT NULL,
    publisher_id BIGINT NOT NULL,
    publisher_name VARCHAR(50) NOT NULL,
    publish_time DATETIME NOT NULL,
    update_time DATETIME
);

-- 7. 创建消息表
CREATE TABLE IF NOT EXISTS message (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    user_id BIGINT NOT NULL,
    title VARCHAR(100) NOT NULL,
    content VARCHAR(500) NOT NULL,
    type VARCHAR(50) NOT NULL,
    is_read VARCHAR(10) NOT NULL DEFAULT '否',
    send_time DATETIME NOT NULL
);
```

## 3. 验证数据库

导入成功后，可以运行以下 SQL 验证：

```sql
USE campus_pet_system;

-- 查看用户
SELECT * FROM users;

-- 查看动物
SELECT * FROM pets;

-- 查看公告
SELECT * FROM announcement;
```

## 4. 修改数据库配置（可选）

如果您的 MySQL 配置不是 `root/root`，请修改 `src/main/java/com/campuspet/utils/DbUtil.java` 中的配置：

```java
private static final String URL = "jdbc:mysql://localhost:3306/campus_pet_system?useSSL=false&serverTimezone=Asia/Shanghai&characterEncoding=UTF-8";
private static final String USERNAME = "root";  // 修改为您的用户名
private static final String PASSWORD = "root";  // 修改为您的密码
```

## 5. 重新运行程序

导入数据库后，重新运行程序即可：

```bash
java -jar target/campus-pet-system-1.0-SNAPSHOT-jar-with-dependencies.jar
```
