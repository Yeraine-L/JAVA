# 校园流浪猫狗投喂与领养管理系统 - 数据库设计文档

## 1. 数据库方案

- 数据库：MySQL 8.0
- 字符集：utf8mb4
- 排序规则：utf8mb4_general_ci
- 连接方式：JDBC
- 数据库名建议：campus_pet_system

## 2. 设计原则

- 表名使用小写下划线风格
- 主键统一使用 bigint 或 int 自增
- 时间字段统一保留 create_time、update_time 或业务时间字段
- 状态字段统一使用 varchar，便于答辩演示和数据查看
- 保留必要的冗余展示字段，减少复杂联表成本

## 3. 表结构总览

## 6. 初始化数据建议

### 6.1 预置账号

- 管理员：admin / admin123
- 普通用户：user01 / 123456
- 志愿者：vol01 / 123456

### 6.2 预置动物

- 至少 6 条动物数据
- 包含猫、狗两类
- 状态覆盖待领养、已领养、救助中

### 6.3 预置公告

- 领养须知
- 投喂注意事项
- 救助活动通知
